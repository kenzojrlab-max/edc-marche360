# EDC Marchés360

Plateforme intégrée de gestion du cycle de vie des marchés publics pour EDC S.A.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
